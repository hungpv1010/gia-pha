🔥 Cấu hình Firebase

Tạo project tại console.firebase.google.com → Project Settings → Your apps → Web app → Copy config

Các bước nhanh:
1. Tạo project Firebase mới
2. Vào Firestore Database → Create database → Start in test mode
3. Vào Project Settings → Add app → Web → Copy firebaseConfig
4. Dán các giá trị vào bên dưới
